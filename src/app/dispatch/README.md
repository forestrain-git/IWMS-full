## 功能特性

### 1. AI智能调度大脑
- **实时任务统计**：车辆、任务、应急事件、AI建议的实时统计
- **AI优化建议**：路径优化、车辆重新分配、油耗优化等智能建议
- **一键应用方案**：AI建议可一键应用到实际调度中
- **置信度评估**：每个AI建议都有置信度评分

### 2. 一张图调度指挥
- **GIS地图显示**：基于高德地图的实时车辆和任务可视化
- **实时车辆轨迹**：5秒更新频率，显示车辆位置、速度、方向
- **任务状态可视化**：不同颜色和图标表示任务状态和优先级
- **框选批量调度**：支持鼠标框选多车进行批量操作
- **右键菜单**：语音/视频/重新派单/紧急召回等快捷操作

### 3. 动态路径优化
- **遗传算法VRP**：自研简化版遗传算法，5秒内出结果
- **多目标优化**：距离、时间、油耗、碳排放四个优化目标
- **实时路况避让**：算法考虑实时交通状况
- **优化结果对比**：显示与之前方案的改善百分比
- **路径可视化**：在地图上显示优化后的路径

### 4. 智能排班
- **人员排班管理**：早班、中班、夜班、灵活班次管理
- **车辆排班管理**：在岗、休息、下班、维保状态管理
- **工时均衡**：自动检测超时和工时不足，提供均衡建议
- **维保提醒**：7天内需要维保的车辆自动提醒
- **技能匹配**：根据人员技能自动分配合适任务

### 5. 应急调度
- **预案库管理**：满溢、故障、天气、事故四种应急预案
- **跨区协同**：支持跨区域资源调配和协同响应
- **备用资源激活**：一键激活备用车辆和人员
- **响应步骤追踪**：实时显示应急响应进度
- **资源需求评估**：自动计算所需车辆、人员、设备

### 6. 实时闭环管控
- **异常自动预警**：满溢、故障、超时等异常自动预警
- **智能派单**：基于距离和负载的智能任务分配
- **任务进度追踪**：实时显示任务执行进度
- **作业质量验收**：支持任务完成质量评估

### 7. 语音视频调度
- **语音通话**：支持与多车辆同时进行语音通话
- **语音消息**：支持录制语音消息发送给选中车辆
- **视频会商**：支持最多4人同时视频会议
- **屏幕共享**：支持屏幕共享功能
- **连接状态监控**：实时显示网络连接状态

## 技术实现

### 核心技术栈
- **React 18**：现代化前端框架
- **TypeScript**：类型安全的开发体验
- **高德地图API**：GIS地图和路径规划
- **Tailwind CSS**：现代化样式系统
- **Zustand**：全局状态管理（只读selectedStationId）

### 关键算法
- **遗传算法VRP**：
  - 种群大小：50
  - 进化代数：100
  - 交叉率：0.7
  - 变异率：0.1
  - 执行时间：3-5秒
- **距离计算**：哈弗辛公式计算地球表面两点间距离
- **多目标优化**：加权法处理距离、时间、油耗、排放

### 数据模拟
- **车辆数据**：12辆车，包含位置、状态、载重、油位等信息
- **任务数据**：25个任务，包含位置、优先级、工作量等
- **AI建议**：3条智能建议，包含收益评估和置信度
- **应急事件**：2个应急事件，包含响应状态和预案
- **排班数据**：15名员工和12辆车的排班信息

### 实时更新
- **数据更新频率**：5秒更新一次
- **车辆位置**：模拟车辆移动和状态变化
- **任务状态**：模拟任务进度和状态流转
- **AI建议**：定期生成新的优化建议
